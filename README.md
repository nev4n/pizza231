## Задание по разработке кода для группы ИС-231

Работа с git
```
запустите Git Bash
перейдите в каталог c:/xampp/htdocs
> cd c:/xampp/htdocs

Если у вас еще нет форкнутого репозитория
- сделайте Fork репозитория Coopteh/pizza231, клонируйте его на компьютер
```
Выполните работу по курсу "Разработка кода".

Закоммитьте и запуште изменения
```
> git status
> git add .
> git status
> git commit -m "Главная страница"
> git push
```
Сдайте работу - создав запрос на изменения Pull Request  
- зайдите на github и создайте Pull Request со своего аккаунта в исходный репозиторий (для аккаунта Coopteh)
